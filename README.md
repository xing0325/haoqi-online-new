# 好奇 Online

学习社区线上版的第一份前端原型。它刻意不是 Notion 的复刻，而是把课程、时间、正在发生的事和同学之间的连接放到同一个轻盈的首页。

## 当前范围

- 首页仪表盘：课表、调课提示、课程动态、信用积分、阅读进度、在线状态。
- 课程入口与“关注流”式课程预览。
- 官方周表与个人计划视图。
- 阅读、做事空间、社区的可点击入口。
- 纯静态示例数据；尚未接入登录、数据库、Google Sheet 或实际权限。

## 本地查看

任意静态服务器均可，例如：

```powershell
npx serve .
```

然后打开命令输出的本地地址。
